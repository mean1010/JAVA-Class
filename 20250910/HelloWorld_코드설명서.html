🏗️ 코드 구조 분석

1. 클래스 선언

public class HelloWorld
  • public: 접근 제한자로, 다른 패키지에서도 접근 가능
  • class: 클래스를 선언하는 키워드
  • HelloWorld: 클래스의 이름 (파일명과 동일해야 함)

2. 기본 생성자

public HelloWorld() {
}
  • 매개변수가 없는 기본 생성자
  • 내용이 비어있어서 특별한 초기화 작업은 수행하지 않음

3. main 메소드

public static void main(String[] var0)
  • public: 외부에서 호출 가능
  • static: 객체 생성 없이 클래스명으로 직접 호출 가능
  • void: 반환값이 없음
  • main: 프로그램의 진입점 메소드명

⚙️ 코드 동작 분석

1. 환영 메시지 출력

System.out.println("안녕하세요, Java의 세계에 오신 것을 환영합니다!");
System.out.println("Hello, World!");

System.out.println()을 사용하여 콘솔에 텍스트를 출력하고 줄바꿈합니다.

2. 변수 선언 및 초기화

String var1 = "Java 학습자";
byte var2 = 20;
  • String var1: 문자열 타입의 변수에 "Java 학습자" 저장
  • byte var2: 바이트 타입의 변수에 20 저장 (나이를 의미)

3. 자기소개 섹션

System.out.println("\n=== 자기소개 ===");
System.out.println("이름: " + var1);
System.out.println("나이: " + var2 + "살");

+ 연산자를 사용하여 문자열과 변수값을 연결합니다.

4. 간단한 계산 섹션

byte var3 = 10;
byte var4 = 5;
int var5 = var3 + var4;

두 개의 byte 타입 변수를 선언하고, 덧셈 결과를 int 타입 변수에 저장합니다.
